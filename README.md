# Mouse Thingy

Automatic mouse mover/clicker/scroller and keyboard presser based on a JSON config.

## Requirements

- Python 3.13+
- macOS may require Accessibility permissions for controlling mouse/keyboard.

## Install

```bash
uv add pydantic pyautogui
```

## Usage

```bash
python main.py --config config.json
```

## Config

See config.json for an example.
